# Exploration Prompts

---

## Original — Open Exploration
*Curiosity-led rabbit holes, learning something you didn't know to ask*

> Could you learn something unexpected about [topic]? Ask me 1-2 key questions first if you need context. Use any tools you have — web search, files, etc. Create something we can look at together if it makes sense.

---

## Variant — Directed Work
*When you have a specific thing to build or fix*

> Could you help me [thing]? Ask me 1-2 key questions first if you need context. Use any tools you have. Create something we can look at together if it makes sense.

---

## Variant — Thinking Partner
*Working through a decision or problem you haven't resolved yet*

> I'm trying to think through [topic]. Challenge my assumptions where you see them. Ask me 1-2 key questions first if you need context. Use any tools that would help ground this in real information. Build something if it makes it clearer.

---

## Variant — Update / Recalibrate
*When something in your setup, habits, or knowledge feels outdated*

> Help me update how I think about / use [thing] — a lot of what I know is probably [X years] out of date. Ask me 1-2 questions first. Use web search if current information matters. Build something interactive if it helps.

---

## Notes
- "Ask me 1-2 key questions" is the most important constraint
- "If it makes sense" on the artifact keeps the format open
- "Learn something unexpected" gives permission to go adjacent — worth keeping in the exploration version
